# QuickList – HTML, CSS e JavaScript

Este projeto é uma aplicação web simples chamada QuickList, criada para permitir adicionar e organizar itens de forma interativa. Ele foi desenvolvido como parte de uma atividade de aprendizado para praticar o uso de **HTML**, **CSS** e **JavaScript** em conjunto, com foco em lógica, manipulação de eventos e atualização dinâmica da interface.

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript

## Funcionalidades

- Inserção de itens em uma lista
- Atualização dinâmica da lista sem recarregar a página
- Interação com o usuário via JavaScript
- Estilização visual com CSS
